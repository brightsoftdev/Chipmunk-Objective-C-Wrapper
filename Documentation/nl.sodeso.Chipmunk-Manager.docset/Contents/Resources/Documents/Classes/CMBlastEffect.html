CMBlastEffect Class Reference

Inherits from NSObject
Declared in CMBlastEffect.h
CMBlastEffect.m

Overview

The explosion effect can be used to force a group of bodies to act like an explosion event took place.

Tasks

Operators

  • - implosion:position:radius:force: Generates an implosion effect.
  • - implosion:position:radius:force:layer:group: Generates an implosion effect.
  • - explosion:position:radius:force: Generates an explosion effect.
  • - explosion:position:radius:force:layer:group: Generates an explosion effect.

Other Methods

  • - perform:position:radius:force:layer:group:explosion: Performs the blast.
  • - performImpExplosion:cmShape:data: Performs the implosion / explosion callback method.
