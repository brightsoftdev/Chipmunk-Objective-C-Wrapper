CMDefaultLoader Class Reference

Inherits from NSObject
Declared in CMDefaultLoader.h
CMDefaultLoader.m

Overview

The CMLoader class can be used to load complete definitions of bodies and shapes with their constraints into the space. It can be done individual (single body at a time) or multimple at the same time.

Tasks

Operations

  • + loadFromConfiguration:cmSpace:configuration: Loads the configuration from the specified .plist and add's it to the space.
