Overview

A poly shape is a shape that consists of a number of vertices that make up the object. Please note to specify the vertices in a clock-wise maner.

Initialization

  • - initWithBody:vertices: Constructs a new poly shape with the specified vertices. The NSArray must be filled with string with the following formatting: {x,y} (including the brackets)
