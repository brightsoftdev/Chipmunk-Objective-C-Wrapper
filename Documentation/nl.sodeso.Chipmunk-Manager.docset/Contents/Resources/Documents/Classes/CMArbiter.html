Inherits from NSObject
Declared in CMArbiter.h
CMArbiter.m

Overview

The CMArbiter is used for collision detection, this class can specify more information about the collision itself.

Tasks

Initialization

  • - initWithCpArbiter: Creates a new arbiter based on the chipmunk arbiter.

Operations

  • - shapeA Returns the first shape.
  • - shapeB Returns the second shape.
  • - setElasticity: Set the new elasticity which will go into effect when this step is over.
  • - setFriction: Set the new friction which will go into effect when this step is over.
  • - numberOfContacts Returns the number of contacts that occured in this collision.
  • - contactPoints Retourns an array containing CMContactPoint objects that describe all the contact points.
  • - areShapesSwapped Flag indicating if the shapes (shapeA and shapeB) are swapped.
  • - isFirstContact Returns YES if this is the first step that the shapes touched.

Other Methods

  • - dealloc Default dealloc method.
