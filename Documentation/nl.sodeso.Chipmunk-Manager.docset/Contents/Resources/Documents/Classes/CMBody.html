Inherits from CMObject : NSObject

CMBody.m

Overview

The body is used to specify an object. A body consists of one or more shapes and joints that make up the complete object.

Operations

  • - applyImpulse:offset: Apply an impulse to this body at a relative offset from the center of gravity. Both the impulse and the offset are in world coordinates. offset is relative to the position of the body, but not the rotation.
  • - applyForce:offset: Apply (accumulate) the force on the body at a relative offset (important!) from the center of gravity. Both the force and offset are in world coordinates.
  • - updateVelocity:damping:dt: Default rigid body velocity integration function. Updates the velocity of the body using Euler integration.
  • - updatePosition: Default rigid body position integration function. Updates the position of the body using Euler integration. Unlike the velocity function, it’s unlikely you’ll want to override this function. If you do, make sure you understand it’s source code as it’s an important part of the collision/joint correction process.
  • - resetForces Zero both the forces and torques accumulated on body.
  • - sleep Force the body to sleep.
  • - wakeUp Force the body to wake up from sleep.
  • - addToSpace Adds this body to the space.
  • - removeFromSpace Remove this body from the space.
