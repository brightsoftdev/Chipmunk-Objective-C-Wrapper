Overview

Base shape class which is used by all the shapes.

Tasks

Other Methods

  • cpShape The Chipmunk cpShape.

Properties

  • - setElasticity: Elasticity of the shape. A value of 0.0 gives no bounce, while a value of 1.0 will give a ìperfectî bounce. However due to inaccuracies in the simulation using 1.0 or greater is not recommended however.
  • - setFriction: Friction coefficient. Chipmunk uses the Coulomb friction model, a value of 0.0 is frictionless.
  • - setCollisionType: A user defined collision type which can be used to identify a type of object on screen. (is used by the collision detection)
  • - collisionType Returns the user defined collision type.
  • - setGroup: Shapes in the same non-zero group do not generate collisions. Useful when creating an object out of many shapes that you don’t want to self collide. Defaults to CPNOGROUP.
  • - setLayer: Shapes only collide if they are in the same bit-planes. i.e. (a->layers & b->layers) != 0 By default, a shape occupies all bit-planes.
  • - setData: Associates an object with this shape, it is not used by Chipmunk but can be used to create a binding between an Sparrow object and a cpShape.
  • - getData Returns the associated object with this body.

Operations

  • - getBody Returns the body associated with this shape.
  • - addToSpace Add this shape to the Chipmunk space.
  • - addToSpaceAsStatic Add this body to the Chipmunk space as a static object.
  • - removeFromSpace Remove this shape from the Chipmunk space. (it will determine by itself if the Shape is static or not)
