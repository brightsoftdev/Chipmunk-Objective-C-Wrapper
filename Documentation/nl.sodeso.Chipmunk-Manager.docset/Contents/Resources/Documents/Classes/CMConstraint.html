Properties

  • - setMaxForce: Sets the maximum force that the constraint can use to act on the two bodies. Defaults to INFINITY.
  • - setMaxBias: Sets the maximum speed at which the constraint can apply error correction. Defaults to INFINITY.
  • - setBiasCoef: Sets the percentage of error corrected each step of the space. (Can cause issues if you donít use a constant time step) Defaults to 0.1.
  • - firstBody Returns the first body that this constraint is attached to.
  • - secondBody Returns the second body that this constraint is attached to.
  • - setData: Associates an object with this constraint, it is not used by Chipmunk but can be used to create a binding between an Sparrow object and a cpConstraint.
  • - getData Returns the associated object with this constraint.
