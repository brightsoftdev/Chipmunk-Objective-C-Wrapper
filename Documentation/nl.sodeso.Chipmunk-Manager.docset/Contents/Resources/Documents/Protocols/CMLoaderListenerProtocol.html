CMLoaderListenerProtocol Protocol Reference

Declared in CMLoaderListenerProtocol.h

Overview

This protocol can be added to an object so it will receive callbacks about when a body, shape or constraint has been constructed and when it has been added to the space.

This allows you to make modifications to the objects or attach a Sparrow object to the body so it runs in the updates for the positions and angles.

Tasks

  • - bodyBeforeAddToSpace: Callback when a body has been created but not yet added to the space.
  • - bodyAfterAddToSpace: Callback when a body has been created and has been added to the space.
  • - shapeBeforeAddToSpace: Callback when a shape has been created and has been attached to the body but not yet added to the space.
  • - shapeAfterAddToSpace: Callback when a shape has been created and has been attached to the body and has been added to the space.
  • - constraintBeforeAddToSpace: Callback when a constraint has been created and has been attached to both the bodies but not yet added to the space.
  • - constraintAfterAddToSpace: Callback when a constraint has been created and has been attached to both the bodies and has been added to the space.
